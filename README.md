# L&K - E-commerce Website

*L&K* is a basic e-commerce website created using HTML, CSS, and JavaScript on the frontend, with PHP and MySQL used for the backend. The website lets users view products, add them to a cart, and place orders. There is also a simple admin section to manage products and orders.

## Features

- User registration and login
- Product display with categories
- Shopping cart and order system
- Admin section to manage products and orders
- Works on both desktop and mobile screens

## Technologies Used

- HTML, CSS, JavaScript
- PHP
- MySQL
